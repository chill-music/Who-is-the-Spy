# Folder: Asset Library (icos)

## Overview
This directory serves as the primary asset library for the application, containing icons, status badges, VIP identifiers, marriage rings, and animated profile effects.

## Subdirectories

### [rings/](./rings/FOLDER_MAP.md)
- **Purpose**: Assets for the marriage system (Wedding Rings).

### [profileEffects/](./profileEffects/FOLDER_MAP.md)
- **Purpose**: Animated GIFs and transient visual effects for high-status profiles.

### [FootballClubLogo/](./FootballClubLogo/FOLDER_MAP.md)
- **Purpose**: Team-specific assets for the Soccer Star game.

### [vip icos/](./vip%20icos/FOLDER_MAP.md)
- **Purpose**: Icons specific to the VIP tiers.

## Key Files

### PWA Icons (`icon-*.png`)
- **Purpose**: Standard app icons for the PWA manifest and mobile home screens.

### Family Badges (`Family Sign[1-5].png`)
- **Purpose**: Visual indicators of a family's activeness level, ranging from level 1 to level 5.

### VIP Status Identifiers (`idvip*.png`)
- **Purpose**: Badge overlays shown on player names and profile cards to indicate VIP rank.

### Status Crowns (`Crown*.png`, `Purple Crown*.png`, `Royal Crown*.png`)
- **Purpose**: High-tier status badges for top-performing or high-paying users.

### Legend Badges (`Legend[1-5].png`, `Ultimate Legend.gif`)
- **Purpose**: The highest tier of achievement badges in the game.
