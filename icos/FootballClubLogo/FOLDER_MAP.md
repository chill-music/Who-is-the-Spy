# Folder: Team Branding (icos/FootballClubLogo)

## Overview
This directory contains the branding assets for various international football clubs used within the "Soccer Star" mini-game.

## Files

- **Clubs**: AC Milan, Bayern München, Barcelona, Juventus, Liverpool, Manchester United, PSG, and Real Madrid.
- **Purpose**: Teams for wagering or prediction within the game.
- **Logic**: Mapped in `SoccerStarGame.js` to identify specific team choices.
- **Dependencies**: Used globally within the `SoccerStarGame` feature.
