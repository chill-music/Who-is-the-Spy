# Folder: Profile Visual Effects (icos/profileEffects)

## Overview
This directory contains high-resolution animated GIFs used as visual overlays for player profiles. These effects are typically exclusive to VIP 10 or legendary status users.

## Files

- **[Volcano Woman.gif](file:///c:/Users/sheha/OneDrive/Desktop/Who-is-the-Spy-og/icos/profileEffects/Volcano%20Woman.gif)**: Animated fire/magma effect.
- **[fire Woman.gif](file:///c:/Users/sheha/OneDrive/Desktop/Who-is-the-Spy-og/icos/profileEffects/fire%20Woman.gif)**: Animated fire-themed aura.
- **[lightning woman.gif](file:///c:/Users/sheha/OneDrive/Desktop/Who-is-the-Spy-og/icos/profileEffects/lightning%20woman.gif)**: Animated electrical aura.

## Logic
These effects are conditionally rendered by `ProfileEffects.js` based on the user's `profileFrame` or `activeEffect` metadata in Firestore.
