# Quickstart: Profile Redesign: Single Page Layout

This redesign merges multiple tabs into a single-page view.

## Quick Actions
- **View Profile**: Click on your avatar or another player's name to open the profile.
- **Scroll**: All sections (Moments, Gifts, Stats, Achievements) are displayed vertically. Use the scrollbar or touch gestures to navigate.
- **Header**: Avatar, Name, and User ID remain fixed at the top (if possible) or at the start of the scroll view.

## Key Section Changes
- **Moments**: Now appear directly below the User ID for faster social interaction.
- **Gifts**: Now follow the Moments list.
- **Stats**: Found after the Gift Wall.

## Visual Order
1. Avatar/Name/Header
2. Tribe Signs/Gender/User ID (Identity)
3. **📸 Moments (Social Feed)**
4. **🎁 Gift Wall (Gallery)**
5. **⚡ General Stats (Win/Loss/Level)**
6. **🎖️ Achievements (Badges)**
