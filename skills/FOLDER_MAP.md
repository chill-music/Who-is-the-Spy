# Folder: Development Skills (skills)

## Overview
This directory contains the core development methodologies, architectural standards, and operational workflows that all AI agents must follow when contributing to the "Who is the Spy" project.

## Subdirectories

### [clean-code-patterns/](./clean-code-patterns/FOLDER_MAP.md)
- **Purpose**: Coding standards.
- **Logic**: Principles for single-responsibility functions, naming conventions, and avoiding "magic numbers" in JavaScript.

### [documentation-skill/](./documentation-skill/FOLDER_MAP.md)
- **Purpose**: Operational workflow.
- **Logic**: Defines the branch naming strategy (`feature/`, `fix/`), commit message formats, and the PR review process.

### [refactoring-specialist/](./refactoring-specialist/FOLDER_MAP.md)
- **Purpose**: Technical debt management.
- **Logic**: Methodology for safe, incremental refactoring of complex React components without breaking existing behavior.

### [self-improving-agent-skill/](./self-improving-agent-skill/FOLDER_MAP.md)
- **Purpose**: Knowledge persistence.
- **Logic**: Procedures for curating the `MEMORY.md` stack and promoting proven patterns into project-wide rules (`CLAUDE.md`).

### [senior-architect-skill/](./senior-architect-skill/FOLDER_MAP.md)
- **Purpose**: System design.
- **Logic**: Framework for large-scale architectural decisions, dependency mapping, and cross-feature integration strategies.
