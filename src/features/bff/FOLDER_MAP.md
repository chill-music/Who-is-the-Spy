# src/features/bff Directory Map

Contains features for building close relationships ("BFFs") and interacting with official system bots.

## Components & Logic

| File | Purpose |
| :--- | :--- |
| **`BFFService.js`** | Data service for creating, managing, and retrieving BFF relationships. |
| **`BFFModal.js`** | Main UI for viewing and managing your active BFFs and pending requests. |
| **`BFFCardModal.js`** | Detailed view of a specific BFF relationship, including shared stats and "Card" visuals. |
| **`BFFRequestItem.js`** | UI component for individual relationship invitations. |
| **`BFFProfileListModal.js`** | Utility for listing potential BFF candidates. |
| **`BFFStripProfile.js`** | A condensed view of a BFF's profile. |
| **`BotChatModal.js`** | Interface for communicating with official system bots (e.g., The Detective, Love Bot). |
