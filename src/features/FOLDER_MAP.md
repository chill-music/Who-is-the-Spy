# Folder: Feature Ecosystem (src/features)

## Overview
This directory is the primary organizational hub for the project's features. Every substantial game system or social mechanism is contained in its own sub-directory to maintain modularity.

## Feature Categories

### Core Application Views
- **[app/](./app/FOLDER_MAP.md)**: The primary views of the application (Lobby, Room, Ranking).
- **[profile/](./profile/FOLDER_MAP.md)**: User identity, stats, and customization components.

### Social & Relationship Systems
- **[bff/](./bff/FOLDER_MAP.md)**: Close friend ("Best Friend Forever") mechanics.
- **[chat/](./chat/FOLDER_MAP.md)**: Global chat management and communication logic.
- **[couples/](./couples/FOLDER_MAP.md)**: Marriage, proposals, and shared registry systems.
- **[family/](./family/FOLDER_MAP.md)**: Large-scale social groups (tribes) and collective activity.
- **[groups/](./groups/FOLDER_MAP.md)**: Modular circle/groups for smaller player associations.

### Game Mechanics & State
- **[game/](./game/FOLDER_MAP.md)**: Core mission and rules logic.
- **[room/](./room/FOLDER_MAP.md)**: Management of the real-time room instance and player synchronization.
- **[lucky-games/](./lucky-games/FOLDER_MAP.md)**: Mini-games and currency-sink mechanics.

### App Lifecycle & UI Resilience
- **[onboarding/](./onboarding/FOLDER_MAP.md)**: The first-run experience and tutorial flow.
- **[pwa/](./pwa/FOLDER_MAP.md)**: Offline support and installation prompts.
- **[tasks/](./tasks/FOLDER_MAP.md)**: Daily player engagement and reward systems.
