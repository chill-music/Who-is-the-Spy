# src/features/groups Directory Map

Manages persistent tribe and public group chat rooms and channel management.

## Components & Logic

| File | Purpose |
| :--- | :--- |
| **`GroupConfig.js`** | Constants and limits for group chat sessions. |
| **`GroupChatModal.js`** | Main UI for tribal or group communication. |
| **`GroupDetailsModal.js`** | Viewing group info, members, and management tools. |
| **`GroupsList.js`** | A directory of available or joined groups. |
| **`GroupsSection.js`** | A higher-level view component for group discovery and navigation. |
