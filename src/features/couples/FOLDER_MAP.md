# src/features/couples Directory Map

Contains the marriage ("Couples") system, including proposals, weddings, and shared status.

## Components & Logic

| File | Purpose |
| :--- | :--- |
| **`CoupleService.js`** | Core data service for creating and managing marriage states in Firestore. |
| **`CoupleConfig.js`** | Configuration for relationship tiers, rings, and benefits. |
| **`CoupleUIComponents.js`** | Individual UI parts like heart-meters and relationship badges. |
| **`CoupleCardModal.js`** | A visual summary card of a couple's status and anniversary. |
| **`ProposalModal.js`** | The interface for sending a marriage invitation. |
| **`IncomingProposalModal.js`** | The interface for receiving and responding to an invitation. |
| **`WeddingHallModal.js`** | A specialized room view for hosting wedding ceremonies. |
| **`RingsShopSection.js`** | The marketplace for purchasing relationship-specific items. |
