# src/components Directory Map

Houses reusable UI components and specialized modals.

## Subdirectories

### `admin/`
Contains the building blocks of the management dashboard.
- **`AdminOverview.js`**: Summary stats and charts.
- **`UserManagementSection.js`**: Tools for staff to manage players.
- **`ReportsSection.js`**: Reviewing and acting on player flags.
- **`TicketsSection.js`**: Customer support system UI.
- **`BanPanelInline.js`**: Core UI for issuing and managing bans.

### `lobby/`
UI elements specific to the main landing view.
- **`LobbyPublicChatBox.js`**: The embedded chat widget seen in the lobby.

### `modals/`
Standardized popups and full-screen overlay components.
- **`PublicChatModal.js`**: Full-screen or bottom-sheet chat interface.
- **`SettingsModal.js`**: App settings (Audio, Language, Account).
- **`VIPCenterModal.js`**: Benefits and levels for the VIP system.
- **`SupportTicketSection.js`**: The user interface for creating support requests.
- **`BrowseRoomsModal.js`**: List of all available game rooms.
- **`TutorialModal.js`**: Introductory guide for new players.
- **`HelpCenterModal.js`**: FAQ and general documentation for players.
- **`LoginRewards.js`**: The daily bonus claiming UI.
